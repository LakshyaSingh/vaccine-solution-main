The client needs to develop a system in which is able to be consistent automation to manage, secure and optimize applications and cloud resources.
The client needs security, automated operations & visibility. As the client modernize and deploy more clusters,
new challenges are introduced:

- Disparate monitoring tools
- Slow delivery & management processes
- Security and quality an afterthought

With CloudPak for multicloud managment (MCM) the client will be able to reduce IT operations expeneses by up to 75% and 
- Dynamically monitor and ensure compliance
- Create visibility across hybrid environment
- Optimize applications, modernization and IT ops.
- Manage hybrid environments
